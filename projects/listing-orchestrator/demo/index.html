Listing Orchestrator

Seed photos become marketplace-ready listings, then publish to Shopify, eBay, and Etsy. When items sell, the seller is notified and buyer contacts are logged for consent-aware marketing. Production path: Cozy Connoisseur Co.

Reproduce locally

No API keys required for this mock path:

cd projects/listing-orchestrator
python -m venv .venv
# activate venv
pip install -r requirements.txt
python run_demo.py
python sync_orders.py
Sample artifacts also live in showcase/samples/ and regenerable under outputs/. Live channel credentials stay in local .env only.
