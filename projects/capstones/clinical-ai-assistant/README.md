# Clinical AI Assistant — RAG / NLP Capstone

UT Austin McCombs **Post Graduate Program in AI & Machine Learning** (Great Learning) — full-code RAG project over the Merck Manuals medical reference.

## Problem

Clinicians face information overload when diagnosing and treating patients. This project builds a **retrieval-augmented generation (RAG)** prototype grounded in trusted medical reference text to support faster, evidence-based answers.

## Stack

- Google Colab (GPU, preferably T4)
- LLM + embeddings for retrieval-augmented Q&A
- Prompt engineering, RAG setup, evaluation, and business observations in-notebook

## Runtime note

Open the notebook in Colab, set **Runtime → GPU**, then run top-to-bottom before regenerating the HTML export.
